## 1. 서비스 소개
AI 기술을 활용한 이미지 태깅 기반의 사진첩 서비스
<br/>
> - 사용자가 사진 업로드 시 자동으로 태그를 지정하여 분류하는 서비스를 제공한다.
> - 사용자가 폴더 생성 시 태그를 기반으로 한 사진 분류의 용이성을 제공한다.
> - 이미지 검색을 통한 유사한 이미지 검색 기능을 제공한다.
> - 갈수록 늘어가는 개인의 사진을 정리하는데 필요한 시간과 수고를 덜어주고자 제작하였다.
